# 房源与资讯双列展示页面说明

## 页面概述

`src/views/house/info/index.vue` 页面已成功改造为双列布局，左侧60%展示房源信息，右侧40%展示资讯管理，为买方用户提供一站式的房源和资讯浏览体验。

## 页面布局

### 左侧房源区域 (60%)
- **搜索面板**：提供地区和价格筛选功能
- **房源卡片网格**：以卡片形式展示房源列表
- **响应式布局**：根据屏幕宽度自动调整卡片数量

### 右侧资讯区域 (40%) 
- **资讯筛选**：按分类筛选资讯内容
- **资讯列表**：以卡片形式展示资讯摘要
- **分页加载**：支持加载更多功能
- **详情弹窗**：点击查看完整资讯内容

## 主要功能

### 房源功能
1. **地区筛选**：北京、上海、辽宁等地区选择
2. **价格筛选**：1000以下、1000-2000、2000-3000、3000以上
3. **房源展示**：网格布局展示房源卡片
4. **详情跳转**：点击房源卡片跳转到详情页面
5. **悬停效果**：鼠标悬停时卡片上浮和图片缩放

### 资讯功能
1. **分类筛选**：支持全部、房产新闻、购房指南、装修知识、政策解读、市场分析
2. **资讯展示**：标题、摘要、分类标签、发布时间、相关标签
3. **内容摘要**：自动截取前80个字符作为摘要
4. **详情查看**：弹窗形式展示完整资讯内容
5. **加载更多**：分页加载，每页10条资讯
6. **交互反馈**：悬停高亮、加载状态提示

## 数据集成

### 房源数据
- 使用 `propertiesApi.getAllProperties()` 获取房源列表
- 自动生成随机图片作为房源展示图
- 支持点击跳转到房源详情页面

### 资讯数据
- 使用 `infoApi.getAllInfo()` 获取资讯列表
- 支持按分类筛选资讯内容
- 分页显示，初始加载10条，支持加载更多

## 样式特色

### 布局设计
- **双列布局**：左侧60%房源，右侧40%资讯
- **响应式设计**：小屏幕下自动变为上下布局
- **统一间距**：20px间距保持页面整洁
- **圆角卡片**：8px圆角营造现代感

### 交互效果
- **悬停动画**：卡片上浮和边框高亮
- **图片缩放**：房源图片悬停时轻微放大
- **加载状态**：资讯区域支持loading状态
- **平滑过渡**：所有动画使用0.3s过渡

### 颜色方案
- **主色调**：#409eff (Element Plus 主题色)
- **文字颜色**：#303133 (主要文字)、#606266 (次要文字)、#909399 (辅助文字)
- **边框颜色**：#e4e7ed (默认边框)
- **背景色**：白色卡片，#f8f9fa 页面背景

## 响应式适配

### 桌面端 (>1200px)
- 标准双列布局
- 房源卡片3-4列网格
- 完整功能展示

### 平板端 (768px-1200px)
- 上下布局，各占50%高度
- 房源卡片2-3列网格
- 保持所有功能

### 移动端 (<768px)
- 上下布局优化
- 房源卡片1-2列网格
- 简化头部筛选
- 优化触摸交互

## 性能优化

### 数据加载
- 房源数据一次性加载（考虑后期优化为分页）
- 资讯数据分页加载，减少初始加载时间
- 图片懒加载（使用随机图片占位）

### 滚动优化
- 独立滚动区域，避免整页滚动
- 自定义滚动条样式
- 平滑滚动体验

### 内存管理
- 合理的数据结构设计
- 避免不必要的响应式数据
- 组件销毁时清理事件监听
